images

Images can help with examples if you scroll down or up you will see loads of images, you might even see some in this section.

These images are very easy to add and use. All you need is to add the img tag with the parameters set.

The img tag is built up from 2 main parts and 3 optional ones, these are the Image Location, Text, Width, Height, and Style.

The Image Location part is where you put the location of it i have all my images in a folder so my location looks like styles/images/example

The Text part is what shows up if the image is unavaliable. This helps because it means people can still know what it is.

The Width and Height parts are simply to change the height or width of the image without editing it specifically.

And finally the Style which changes how it is positioned like "float:right" will make it appear at the far right.
